# Bear Parlor Knowledge Base

The Bear Parlor demo uses a collection of medical and scientific papers as its
knowledge base. Due to copyright restrictions, PDFs are not included in this
repository. Download each paper and place it in the indicated folder before
running the demo.

## Directory structure

```
examples/bear_parlor/
├── CRISPR/
├── DMG/
├── GLP-1/
├── LLMs in Clinical Decision Support/
├── MS/
├── alzheimers/
├── epilepsy/
└── stroke/
```

## Setup

After downloading all papers to the correct folders, run the ingest script to
build the knowledge base index:

```bash
cd examples/bear_parlor
python ingest.py
```

This will process the PDFs and build the ChromaDB index in `panel_data/knowledge/`.
