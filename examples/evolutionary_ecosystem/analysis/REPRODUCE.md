# Reproducing All Paper Figures and Tables

This document describes how to reproduce every figure and table in the paper
"Behavioral Evolution via Retrieval-Augmented Instruction Corpora."

Run all scripts from the **repository root**.

---

## Dependencies

```bash
pip install matplotlib numpy scipy sentence-transformers torch
```

The embedding model (BAAI/bge-base-en-v1.5) is downloaded automatically on
first run via `sentence-transformers`.

---

## Required Data Files

Place these files in the repository root before running:

### Headless eval results (produced by eval_combined_v2.py)
- `eval3_v2_results_abundance_ice_age.json`
- `eval3_v2_results_famine.json`
- `eval3_v2_results_ice_age.json`
- `eval3_v2_results_predator_bloom.json`
- `eval3_v2_results_expansion.json`
- `eval4_v2_results_merged_final.json`

### Live sim logs
- `sim_log_mendelian_haploid.json`         — Mendelian haploid, N=1744 births
- `sim_log_diploid_codominant.json`        — Diploid co-dominant, N=1202 births
- `sim_log_llm_synthesis_free_epoch.json`  — LLM blend free epoch, N=45 births

### Action log sim data (instrumented runs)
- `sim_log_mendelian_haploid_action.json`      — Mendelian with action+death log
- `sim_log_diploid_codominant_action.json`     — Diploid with action+death log

### LLM blend runs (for mating drift)
- `sim_log_llm_synthesis_abundance_locked.json`

## Reproduce Everything

```bash
python examples/evolutionary_ecosystem/analysis/reproduce_all.py
```

This runs all scripts in order and saves all figures to `paper_figures/`.

## Regenerating Headless Eval Data from Scratch

If you want to rerun the headless evals rather than use the provided result files:

```bash
# Eval 3+4 (all 5 epochs, locus recombination, haploid)
python examples/evolutionary_ecosystem/eval/eval_combined_v2.py \
    --epochs abundance_ice_age famine ice_age predator_bloom expansion \
    --ploidy haploid \
    --base-url http://localhost:8355/v1 --model gemma-4-e2b

# Merge eval4 results
python examples/evolutionary_ecosystem/eval/eval_combined_v2.py --merge-only
```

Requires a running vLLM server:
```bash
python examples/evolutionary_ecosystem/serve_llm.py
```
